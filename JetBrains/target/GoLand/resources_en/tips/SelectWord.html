[55/157] SelectWord(选择单词) [汉化反馈]

To extend or shrink the selection at the caret, press &shortcut:EditorSelectWord; (Edit | Extend Selection) or &shortcut:EditorUnSelectWord; (Edit | Shrink Selection). Using this action multiple times will select the function name, then the expression that calls that function, then the whole statement, then the containing block, etc.
