[129/173] RunRakeTask(运行 rake 任务) [汉化反馈]

The easiest way to run a Rake task is to choose Run Rake task in the Tools menu, or to press &shortcut:org.jetbrains.plugins.ruby.tasks.rake.actions.RakeTasksPopupAction;.

As you type the name of the task you want to run, the lookup list shrinks to show the matching entries only.
