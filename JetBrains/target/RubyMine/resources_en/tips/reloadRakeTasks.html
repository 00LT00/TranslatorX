[114/173] reloadRakeTasks(reloadrake tasks) [汉化反馈]

If you have created a Rake task and still can't see it in the list, choose Tools | Run Rake Task in the main menu, or press &shortcut:org.jetbrains.plugins.ruby.tasks.rake.actions.RakeTasksPopupAction;.

Then, in the pop-up window, choose [Reload Rake tasks list].
